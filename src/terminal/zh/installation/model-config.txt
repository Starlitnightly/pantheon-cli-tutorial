# 模型配置Example

## 1. 配置 OpenAI API 密钥
$ /api-key openai sk-xxxxxxxxxxxxxxxxxxxx
✓ OpenAI API密钥已成功配置

## 2. 配置 Claude API 密钥  
$ /api-key anthropic sk-ant-xxxxxxxxxxxxxxxxxxxx
✓ Claude API密钥已成功配置

## 3. 设置默认模型
$ /model gpt-5
✓ 默认模型已设置为: gpt-5

## 4. 查看当前配置
$ /api-key
