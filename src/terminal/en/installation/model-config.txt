# Model Configuration Command Demonstration

## 1. Configure OpenAI API Key
$ /api-key openai sk-xxxxxxxxxxxxxxxxxxxx
✓ OpenAI API key successfully configured

## 2. Configure Claude API Key  
$ /api-key anthropic sk-ant-xxxxxxxxxxxxxxxxxxxx
✓ Claude API key successfully configured

## 3. Set Default Model
$ /model gpt-5
✓ Default model set to: gpt-5

## 4. View Current Configuration
$ /api-key
