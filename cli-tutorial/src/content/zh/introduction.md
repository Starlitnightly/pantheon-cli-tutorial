# Pantheon-CLI 介绍

欢迎来到 Pantheon-CLI 的教程！

## 什么是 Pantheon-CLI？

Pantheon-CLI是一个为数据科学设计的开源Agent工具。我们可以通过pantheon-CLI完成几乎所有的数据科学分析任务，通过自然语言与编程语言的混合式编程，Pantheon-CLI可以让任何数据科学家用任何方式探索他们的数据。

## 为什么使用 Pantheon-CLI？

- **无学习门槛**：你可以在10min内掌握Pantheon-CLI的全部用法
- **自由探索数据**：你不需要再去寻找他人帮助来分析数据
- **数据本地化**：分析全在本地进行，完全不用担心数据隐私问题
- **高度定制化**：Pantheon-CLI是完全开源的，你不用担心其有所限制
