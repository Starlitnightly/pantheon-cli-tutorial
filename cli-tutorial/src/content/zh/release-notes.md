# 发布日记

## v0.3.0 (2024-12-XX)

### 新功能
- 新增混合编程支持，支持 Python/R/Julia 无缝切换
- 添加变量持久化功能，变量在不同语言间直接共享
- 实现"氛围分析"框架，智能理解用户研究意图

- 优化 CLI 响应速度，提升用户体验
- 改进错误提示信息，更加清晰易懂
- 增强本地模型支持，提升数据隐私保护

### 修复
- 修复在某些系统上安装失败的问题
