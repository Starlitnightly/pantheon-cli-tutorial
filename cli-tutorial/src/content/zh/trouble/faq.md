# 常见问题 FAQ

本页面回答使用Pantheon-CLI教程时的常见问题。

## 基础问题

### Q: 什么是Pantheon-CLI教程？
**A:** 这是一个交互式的网页教程，帮助用户学习生物信息学、数据科学和命令行工具的使用。您可以在浏览器中直接运行命令，无需安装任何软件。

### Q: 我需要安装什么软件吗？
**A:** 不需要！这是一个纯网页教程，您只需要一个现代浏览器（Chrome、Firefox、Safari或Edge）即可开始学习。

### Q: 教程是免费的吗？
**A:** 是的，完全免费开源。

## 技术问题

### Q: 为什么终端加载很慢？
**A:** 可能的原因和解决方案：
- **网络连接慢**：检查网络连接，尝试刷新页面
- **浏览器性能**：关闭其他标签页，清除浏览器缓存
- **浏览器兼容性**：使用推荐的现代浏览器

### Q: 复制粘贴不工作怎么办？
**A:** 尝试以下方法：
```bash
# 标准复制粘贴
Ctrl+C / Ctrl+V (Windows/Linux)
Cmd+C / Cmd+V (macOS)

# 如果不工作，尝试：
Ctrl+Shift+C / Ctrl+Shift+V
右键菜单选择复制/粘贴
```

### Q: 终端命令没有响应？
**A:** 检查步骤：
1. 确认命令拼写正确
2. 等待几秒钟（某些命令需要时间执行）
3. 按 `Ctrl+C` 取消当前命令
4. 刷新页面重新开始

### Q: 能在移动设备上使用吗？
**A:** 可以，但我们推荐使用桌面浏览器获得最佳体验。移动设备的限制：
- 屏幕较小，查看代码不便
- 键盘输入体验较差
- 某些功能可能不完全支持

## 学习问题

### Q: 我是编程新手，能跟上教程吗？
**A:** 可以！教程从基础开始，循序渐进：
1. 从 "介绍" 部分开始
2. 按顺序学习各个章节
3. 动手实践每个示例
4. 遇到问题查看故障排除页面

### Q: 需要什么背景知识？
**A:** 建议具备：
- 基本的计算机操作能力
- 对数据分析有一定兴趣
- 高中数学水平

不需要：
- 编程经验（教程会教您）
- 生物学背景（会解释相关概念）
- 统计学专业知识（会逐步介绍）

### Q: 学完教程大概需要多长时间？
**A:** 取决于您的学习节奏：
- **快速浏览**：2-3小时
- **认真学习**：1-2周（每天1-2小时）
- **深入掌握**：1-2个月（包括实践项目）

### Q: 可以跳过某些章节吗？
**A:** 可以，但建议：
- **必学**：介绍、安装、基础命令
- **重要**：高级用法、实际案例
- **可选**：特定领域的案例（如分子对接）

1. 使用教程中提到的公开数据集
2. 访问相关的数据库网站
3. 参考教程中的数据获取方法

### Q: 案例的难度如何？
**A:** 案例分为不同难度：
- **初级**：社会行为分析、财务数据分析
- **中级**：RNA-seq分析、混合编程
- **高级**：单细胞分析、分子对接

## 错误和故障

### Q: 遇到JavaScript错误怎么办？
**A:** 解决步骤：
1. 打开浏览器开发者工具（F12）
2. 查看Console中的错误信息
3. 尝试禁用浏览器扩展
4. 使用无痕模式测试
5. 更新浏览器到最新版本

### Q: 页面加载不完整？
**A:** 可能的解决方案：
1. 清除浏览器缓存和cookies
2. 检查网络连接
3. 尝试不同的浏览器
4. 关闭VPN或代理（如果使用的话）

### Q: 命令结果与教程不符？
**A:** 检查以下几点：
1. 确认命令拼写正确
2. 检查是否在正确的目录
3. 确认数据文件存在
4. 查看是否有错误信息
