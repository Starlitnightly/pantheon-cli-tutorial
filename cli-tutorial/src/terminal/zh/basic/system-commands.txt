$ /bio

?? Bio Analysis Tools
/bio list - List all available bio analysis tools
/bio info <tool> - Get information about a specific tool
/bio help  - Get help for bio tools
/bio <tool> <command> - Run tool-specific commands

Examples:
  /bio list                      # Show all available tools
  /bio atac init                 # Initialize ATAC-seq project
  /bio atac upstream ./data      # Run upstream ATAC analysis
  /bio scatac init               # Initialize scATAC-seq project
  /bio scatac upstream ./data    # Run cellranger-atac analysis
  /bio scrna init                # Initialize scRNA-seq project
  /bio scrna analysis ./data    # Load and analyze scRNA-seq data
  /bio rna init                 # Initialize RNA-seq project
  /bio rna upstream ./data      # Run RNA-seq upstream analysis
  /bio spatial init              # Initialize spatial project
  /bio spatial run_spatial_workflow <workflow_type> # Run spatial workflow
  /bio dock init                # Initialize molecular docking project
  /bio dock run_dock            # Interactive molecular docking workflow
  /bio dock run ./data          # Run batch molecular docking on folder
  /bio GeneAgent TP53,BRCA1,EGFR # Gene set analysis with AI
